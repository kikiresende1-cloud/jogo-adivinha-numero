# 🎮 Jogo: Adivinha o Número

## 📌 Descrição

Este projeto consiste num jogo interativo desenvolvido em Python utilizando a framework Streamlit.

O objetivo do jogo é adivinhar um número secreto gerado aleatoriamente pelo sistema dentro de um determinado intervalo.

O utilizador recebe pistas indicando se o número secreto é maior ou menor que o valor introduzido e tem um número limitado de tentativas para acertar.

---

## ⚙️ Tecnologias Utilizadas

- Python
- Streamlit
- Visual Studio Code

---

## 🎯 Funcionalidades

- Interface web interativa
- Geração aleatória de número
- Sistema de pistas (maior / menor)
- Contador de tentativas
- Níveis de dificuldade
- Limite de tentativas
- Botão para reiniciar o jogo
- Animação de vitória

---


## 🕹️ Como executar o projeto

1. Instalar o Streamlit

```
pip install streamlit
```

2. Executar o jogo

```
python -m streamlit run app.py
```

3. Abrir no navegador

```
http://localhost:8501
```
